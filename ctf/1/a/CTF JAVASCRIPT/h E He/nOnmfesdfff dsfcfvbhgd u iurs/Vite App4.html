Les Boucles

Une boucle permet de répéter un certain nombre de fois une action dans un programme. Celles-ci peuvent être représentées de plusieurs manières mais dans notre cas, nous nous intéresserons au "while".
        
            // Imaginons que nous avons un nombre que nous voulons afficher dans la console en incrémentant à chaque affichage jusqu'à 100
            
            // Nous aurons alors comme code : 
            var i = 0;
            while (i < 100){
              console.log(i); // console.log() permet d'afficher en console ce qu'il se trouve entre ses parenthèses 
              i = i+1;
            }
