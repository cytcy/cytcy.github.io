Les variables

Une variable est quelque chose servant à stocker une valeur, parfois modifiable par la suite, à l'aide d'un nom.
         
            Exemple :
            var monChiffre = 2;
Ceci permet de stocker la valeur 2 dans une variable nommée "monChiffre". Celle-ci peut-être appelée plus tard afin de récupérer la valeur étant dedans.
        
            Exemple : 
                monChiffre
            //renverra
                2
Les variables peuvent aussi être réattribuées pour changer la valeur qu'elles contiennent.
        
          Exemple :
            var monChiffre = 2;
            // monChiffre vaut 2
            monChiffre = 4;
            // maintenant il vaut 4
